# サイコロ対戦ゲーム (チンチロリン風 Webアプリ)　※料金の観点でEC2にはあげていないため、ローカル操作
※ポイント機能実装途中　※個人製作

Flaskを利用して作成された、2人対戦型（プレイヤー同士、またはプレイヤー vs CPU）のサイコロゲームです。伝統的な「チンチロリン」のルールをベースにしており、サイコロ3つの出目によって役と勝敗が決まり、掛け金（Bet）に応じてポイントが変動します。

## 使用技術
# 言語
* python3
* HTML/CSS
* JavaScript(アニメーション)
# フレームワーク
* Flask

## 🎮 機能概要

* **役の判定**: ピンゾロ、456、ゾロ目などの役を自動判定します。
* **掛け金と倍率計算**: 勝敗と上がった役に応じて、獲得ポイントの倍率が変動します。
* **非同期通信 (AJAX)**: 画面遷移なしでサイコロを振り、リアルタイムに結果とポイントを更新します。

## 🎲 役とスコアのルール

サイコロを3つ振り、以下の組み合わせによってスコアと倍率が決定します。
# ※ ポイント機能実装途中

| 役の名前 | サイコロの目 | スコア | 勝利時の掛け金倍率 |
| :--- | :--- | :--- | :--- |
| **ピンゾロ** | `1, 1, 1` | 100 | x 5.0 |
| **ゾロ目** | `2,2,2` など | 80 | x 3.0 |
| **456だぜ！** | `4, 5, 6` | 60 | x 2.0 |
| **払い戻し** | 2つの目が一致 | 50 | x 1.0 |
| **目無しだぜ！** | 上記以外のバラバラ | 10 | (倍率適用なし) |
| **123だぜ！** | `1, 2, 3` | 0 | x 0.5 |

> **Note**: 負けた場合、または引き分けの場合はポイントが没収（-1.0倍）されます。両プレイヤーは初期ポイント `1000` からスタートします。

## 🛠 必須環境

* Python 3.x
* Flask

## 🚀 セットアップと起動方法

### 1. リポジトリをクローンまたはダウンロードします。
### 2. 必要なライブラリ（Flask）をインストールします。
   bash
   ```
   pip install Flask
   ```
### 3. アプリケーションの起動
ターミナルで、app.py を保存したフォルダまで移動し、以下のコマンドを実行してサーバーを立ち上げます。
Bash
```
python app.py
```
http://localhost:5000にアクセス
